5  KNWRobot.h - Library written to control robots for SMU KNW2300
6  Controling Arduino: MEGA 2560
7  Author: Alexandria Hancock
8 
9  When first starting to interface with the MEGA 2560 to get the robot
10  runnning, a couple things must first be wired so you can use the library.
11  The numpad, which must be on pins {39,41,43,45,47,49,51,53}, the
12  Adafruit PWM board, and the LCD 16x2 must all be correctly wired to check
13  if you have successfully included the library in your sketch, if you have,
14  the message "SMU Lyle KNW2300" will appear on the first line of the lcd.
15 
16  To include this library in your sketch, you have to make a folder in the
17  libraries folder in the arduino files. For example, for me I keep it in
18  the c drive in my Program Files folder. The folder must have the name
19  KNWRobot, and will have the .h, the .cpp, as well as a README.txt within it.
20  The path should look sommething like this:
21 
22  C:\Program Files (x86)\Arduino\libraries\KNWRobot
23 
24  After this step is done, open up a sketch in the Arduino IDE and type
25  #include <KNWRobot.h>
26 
27  This will give you access to run all of the functions we've written to safely
28  run your robot. Keep in mind, you do have the freedom to edit the library
29  however you see fit (for example changing the pins for the numpad), but if you
30  run into errors that you can't fix, we will tell you to delete the edited library
31  and redownload it for it's original state.
