Detailed Description

A struct representing the component of a Motor / Servo, used to send signals to the PWM.

ID - the user-defined ID for the motor PIN - The physical pin where the motor is put into the PWM TYPE - 'a' for analog, 'd' for digital, and 'p' for PCA board ZERO - The origin of the servo / motor OBJ - The Servo object instance used by the Arduino
